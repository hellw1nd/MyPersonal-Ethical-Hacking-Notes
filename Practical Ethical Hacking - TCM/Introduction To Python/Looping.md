### For Loop
```python
#For loops - start to finish of an iterate
vegetables = ["cucumber", "spinach", "cabbage"]
for x in vegetables:
	print(x)
	
```
### While Loops
```python
#While loops - execute as long as true
i = 1

while i < 10:
	print(i)
	i += 1
```
